# FreshKeeper

FreshKeeper is a Flutter-based mobile application for managing fruits and vegetables.

## Features

- Login
- Category Management
- Vegetable/Fruit Management
- SQLite Database (sqflite)
- CRUD Operations
- Shelf Life Tracking
- Freshness Status
- Weekly & Monthly Reports

## Technologies

- Flutter
- Dart
- SQLite (sqflite)
- SharedPreferences
- Material Design
